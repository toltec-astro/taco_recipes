PLEASE READ
==========

Observations of one molecular cloud core.

Source: Rho Oph A
RA: 16h 26m 27.5s
DEC: -24d 23' 57"

On nights when no HWP is installed, we should use the script for Stokes I.  

On nights when the HWP is installed and spinning, we should use the scripts for StokesQU.

Stokes I: 
---------
Pattern type: 10 min 4x4 Lissajous mapping pattern  
File: RhoOphA_StokesI.lmtot

Execute this 6 times.  Total time: ~ 60 minutes


Stokes Q/U:
-----------

4x4 rasters with 5 arcsec/s mapping velocity and 0.40 arcmin spacing 
(~9 min)

There are 2 scripts with 2 different rotations: 0 and 90

Files: 

RhoOphA_StokesQU_rot0_processed.lmtot

RhoOphA_StokesQU_rot90_processed.lmtot

Execute each script 6 times, alternating between them.  Total time: ~ 1 hour 50 min

Availability: 
-----------

On April 26, Rho Oph A will be above 30 degrees elevation between 5:40 UT and 11:30 UT
