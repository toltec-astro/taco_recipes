Observations of 3c286.  This is an unresolved quasar.

Source: 3C286
RA: 13h 31m 8.288s
DEC: 30d 30' 32.96s

On nights when no HWP is installed, we should use the script for Stokes I.  

On nights when the HWP is installed and spinning, we should use the scripts for StokesQU.

Stokes I: 
---------

Pattern type: 5 min 3x3 Lissajous mapping pattern  
File: target_3C286_Stokes_I.lmtot

Execute this file 5 times.  Total time: ~ 5 x 5 ~ 25 min

Stokes Q/U:
-----------

3x3 rasters with 5 arcsec/s mapping velocity and 0.23 arcmin spacing 
(~9 min give or take depending on turnaround time)

There are scripts with 6 different rotations: 0, 30, 60, 90, 120, and 150

Files: target_3C286_QU_rotXX_processed.lmtot

Execute each of them 1 time. Total time: ~ 6 x 9 = 55 min

Availability: 
-----------

On April 26th, 3c286 is above 30 degrees between 1:30 UT and 10:00 UT.
